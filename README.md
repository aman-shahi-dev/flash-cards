# Flash Cards

This is a solution to the [Flash Cards project](https://roadmap.sh/projects/flashcards) on [roadmap.sh](https://roadmap.sh/).

PROJECT URL: https://roadmap.sh/projects/flash-cards

## Overview

A simple, interactive flashcards application built with React to help users test their JavaScript knowledge. The application features a clean UI with progress tracking, 3D flip animations for revealing answers, and sequential navigation through a predefined set of questions.

### Tech Stack
* **React** (Frontend library)
* **Tailwind CSS** (Utility-first CSS framework for styling)
* **Vite** (Build tool and development server)
* **ESLint** (Code quality)
* **Tabler Icons** (UI icons)

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) and npm (Node Package Manager) installed on your machine.

### Installation

1. Clone the repository:
 ```bash
  git clone [https://github.com/aman-shahi-dev/flash-cards.git](https://github.com/aman-shahi-dev/flash-cards.git)
```
Navigate into the project directory:

 ```bash
cd flash-cards
```
Install the project dependencies:

 ```bash
npm install
```
Running the Development Server
To start the local development server with Vite, run:

 ```bash
npm run dev
```
This will start a local server (usually at http://localhost:5173/). Any changes you make to the files will automatically reflect in the browser.
